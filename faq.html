Frequently Asked Questions



Below ↓ you might see some answers for your questions. You cannot find an answer? Join the support server.

How to invite Chia?



You can invite Chia by clicking here.

I can't hear Chia.


Make sure bot is not Server Muted.

Change the server voice region to any country your close to.

Type the command .disconnect & .play, try again.

If problem still occurs please join the support server.

Chia sounds laggy.


Change the server voice region to any country your close to.

Type the command .disconnect & .play, try again.

If problem still occurs please join the support server.

Chia is not responsive.


Check if Chia is online.

Make sure Chia has the permissions View Channels, Send Messages, Embed Links.

Make sure that you are using the correct prefix. To know the server prefix simply type @Chia

If problem still occurs please join the support server.

I discovered a bug.


Please join the support server and open a ticket at #help-desk

How do I activate free 24/7?


Make sure Chia has the permissions View Channels, Send Messages, Embed Links, Connect.

Simply summon the bot to your voice channel by typing .join

And type the command .247

If you encounter any more problems please join the support server.
